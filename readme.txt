Create Simple Responsive Slide Menu For Mobile, Use shortcode [responsiveslidemenu] in header.php
== Description ==
This plugin adds Responsive Mobile Menu functionality to WordPress.LR Responsive Slide Menu has multi level menu support.Cross browser Compatible.Combine multiple menus.For more features Please contact us at [Logicrays](http://www.logicrays.com/contact-us-logicrays-india/)
== Installation ==1. Upload the `responsiveslidemenu` folder to the `/wp-content/plugins/` directory1. Activate the plugin through the 'Plugins' menu in WordPress1. Place `<?php do_shortcode('[responsiveslidemenu]'); ?>` in your header.php
